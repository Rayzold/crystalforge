Vehicle artwork folder

Drop vehicle images into this folder using these primary paths:

./assets/images/vehicles/Scout Buggy.png
./assets/images/vehicles/Trail Buggy.png
./assets/images/vehicles/Siege Buggy.png
./assets/images/vehicles/Elemental Skiff.png
./assets/images/vehicles/Elemental Frigate.png
./assets/images/vehicles/Grand Elemental Airship.png

The UI will also try these extensions automatically if the .png file is missing:
.jpg
.jpeg
.webp
.svg
