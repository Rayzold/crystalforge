Drop building artwork here.

Suggested approach:

- Name files after the building key, for example:
  - `Farmlands.png`
  - `Arcana Tower.png`
  - `Crystal Upgrade__Epic.png`

You can also assign a custom image path in the admin console for each building.

Supported common formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.svg`
